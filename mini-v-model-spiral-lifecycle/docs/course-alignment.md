# Course Alignment -- V-Model & Spiral Lifecycle

## MIT 16.842: Fundamentals of Systems Engineering
- **V-Model**: Requirements -> Design -> Implementation -> Verification -> Validation
- **Traceability**: Left-to-right and right-to-left traceability

## CMU 24-654: Systems Thinking
- **Risk management**: Boehm's spiral model as risk-driven development
- **Gate reviews**: Decision gates at each lifecycle phase

## Key Concepts
```
V-Model phases: Requirements -> Architecture -> Design -> Code
                -> Unit test -> Integration test -> System test -> Acceptance test

Spiral phases: Determine objectives -> Identify risks -> Develop & test -> Plan next

Boehm's risk exposure: RE = P(loss) * S(loss)
```

## Exercises
1. Trace a requirement through the V-model
2. Compute risk exposure for a spiral iteration
3. Compare V-model vs. spiral model for a given project
